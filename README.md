
### Project Description:
-----
Fast API Project for CRUD Operation & API serving using sqlalchemy. <br>
DB: PostgreSQL- 17.4


### Installation & Run: 
-----
Rename .env.example to .env

```
docker-compose build 
```

Then 

```
docker-compose up   
```
or to run demon mode

```
docker-compose up   -d
```
